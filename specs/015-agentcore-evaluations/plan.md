# Implementation Plan: AgentCore Evaluations

**Branch**: `015-agentcore-evaluations` | **Date**: 2026-03-18 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/015-agentcore-evaluations/spec.md`

## Summary

Enable AgentCore Evaluations for the strands-demo agent by configuring 4 built-in evaluators (GoalSuccessRate, Helpfulness, Correctness, Faithfulness) and 1 custom tool-selection evaluator using LLM-as-a-judge. Provide a CLI wrapper script for common evaluation operations. The evaluation execution role is auto-created by the CLI — no changes to the existing CloudFormation template are needed.

## Technical Context

**Language/Version**: Bash (shell script), JSON (evaluator config) — no Python code changes
**Primary Dependencies**: `bedrock-agentcore-starter-toolkit` CLI (`agentcore` command, already installed)
**Storage**: CloudWatch Logs (evaluation results written to `/aws/bedrock-agentcore/evaluations/results/{config-id}`)
**Testing**: Manual validation via `agentcore eval run` and `agentcore eval evaluator list`
**Target Platform**: Linux/macOS development environment with AWS CLI configured
**Project Type**: Tooling/infrastructure — shell scripts and JSON config files
**Performance Goals**: On-demand evaluation completes in under 2 minutes
**Constraints**: No agent code changes (FR-006), use agentcore CLI only (FR-008)
**Scale/Scope**: Demo project with low traffic — 100% sampling rate

### Built-in Evaluators

| Evaluator ID | Level | What It Measures |
|-------------|-------|-----------------|
| Builtin.GoalSuccessRate | SESSION | Whether the conversation achieved the user's goals |
| Builtin.Helpfulness | TRACE | How useful and valuable the response is from user's perspective |
| Builtin.Correctness | TRACE | Whether information in the response is factually accurate |
| Builtin.Faithfulness | TRACE | Whether response is grounded in tool results (not hallucinated) |

### Custom Evaluator: tool_selection_accuracy

| Property | Value |
|----------|-------|
| Name | tool_selection_accuracy |
| Level | TRACE |
| Judge Model | global.anthropic.claude-sonnet-4-5-20250929-v1:0 |
| Rating Scale | 0.0 (Incorrect), 0.5 (Partial), 1.0 (Correct) |
| Evaluates | Whether the agent selected appropriate tools for the user's request |

### CLI Wrapper Subcommands

| Subcommand | Description | Underlying CLI |
|-----------|-------------|---------------|
| `run <session-id> [--days N]` | On-demand evaluation with all evaluators | `agentcore eval run --evaluator ... --session-id ...` |
| `setup` | Create/update online evaluation config at 100% sampling | `agentcore eval online create ...` |
| `list` | List all online evaluation configs | `agentcore eval online list` |
| `status <config-id>` | Get details of a specific config | `agentcore eval online get --config-id ...` |
| `create-evaluator` | Create the custom tool_selection_accuracy evaluator | `agentcore eval evaluator create ...` |
| `list-evaluators` | List all available evaluators | `agentcore eval evaluator list` |
