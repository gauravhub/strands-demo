# Implementation Plan: Python Environment Bootstrap

**Branch**: `001-python-env-bootstrap` | **Date**: 2026-03-09 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/001-python-env-bootstrap/spec.md`

## Summary

Bootstrap the project's Python environment using `uv` as the package manager. Deliverables
are: `pyproject.toml` (project metadata + all dependency declarations), `uv.lock`
(committed lock file for reproducibility), `.python-version` (uv-managed Python pin),
`.env.example` (secrets template for AWS + Cognito credentials), and `README.md`
(setup instructions). No application code is produced in this feature.

## Technical Context

**Language/Version**: Python 3.11+
**Primary Dependencies**: `strands-agents`, `strands-agents-tools`, `streamlit`, `boto3`,
`python-dotenv` (runtime); `pytest` (dev)
**Storage**: N/A
**Testing**: `pytest` (dev dependency; no tests authored in this feature)
**Target Platform**: Linux / macOS / WSL2
**Project Type**: Application (non-published; no `[build-system]` block in pyproject.toml)
**Performance Goals**: Full `uv sync` from clean clone completes in under 5 minutes
**Constraints**: `uv` must be pre-installed by developer; Windows native not supported
**Scale/Scope**: Single developer machine setup; lock file ensures cross-machine consistency

introduces application code. This is intentional (Principle I: Simplicity First).

## Implementation Notes

### `pyproject.toml` shape

```toml
[project]
name = "strands-demo"
version = "0.1.0"
description = "AWS Strands Agents demo with Streamlit frontend and Cognito auth"
readme = "README.md"
requires-python = ">=3.11"
dependencies = [
    "strands-agents>=0.1.0",
    "strands-agents-tools>=0.1.0",
    "streamlit>=1.35.0",
    "boto3>=1.34.0",
    "python-dotenv>=1.0.0",
]

[dependency-groups]
dev = [
    "pytest>=8.0",
    "pytest-cov>=5.0",
]
```

### `.env.example` content

All AWS core variables + full Cognito variable set (see `research.md` Decision 3).
Every variable has a placeholder value and inline comment describing its purpose.

### `README.md` content

Must cover: prerequisites (uv install), setup commands (`uv sync`), environment
configuration (`.env.example` → `.env`), and verification command (import check).
Reference `specs/001-python-env-bootstrap/quickstart.md` for full validation steps.
